使用步骤

  1. 先点上方 唤醒 / 重新检测，直到状态为绿色「在线」
  2. 安装 Tampermonkey 后，点击 安装 bootstrap
  3. 进入三国杀 OL 页面，脚本会自动连接服务器并加载完整小抄
  4. 若提示冷启动，等待约 30 秒后刷新游戏页；或使用缓存版继续

更新小抄：在本地修改 幽灵山庄小抄.txt 后运行 npm run build 并部署； 玩家刷新游戏页即可获取新模块（bootstrap 会校验 sha256）。
